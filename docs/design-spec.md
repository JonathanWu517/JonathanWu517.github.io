# 设计规范

## 色彩系统

### 深色模式（默认）

| 色阶 | 色值 | 使用场景 |
|------|------|----------|
| 页面背景 | `#1e1e1e` | 最底层背景 |
| 侧边栏/次要背景 | `#252526` | 导航栏、侧边栏 |
| 卡片背景 | `#2d2d30` | 卡片、面板、列表项 |
| 主要文字 | `#cccccc` | 标题、正文 |
| 次要文字 | `#9d9d9d` | 描述、辅助信息 |
| 辅助文字 | `#6a6a6a` | 日期、元信息 |
| 主强调色 | `#60a5fa` | 链接、按钮背景、标题高亮 |
| 次强调色 | `#3b82f6` | 按钮悬停、标签文字 |
| 浅强调色 | `#93c5fd` | 装饰元素 |
| 分割线 | `rgba(255,255,255,0.08)` | 卡片边框、分割线 |

### 浅色模式

| 色阶 | 色值 | 使用场景 |
|------|------|----------|
| 页面背景 | `#ffffff` | 最底层背景 |
| 次要背景 | `#f3f4f6` | 导航栏、侧边栏 |
| 卡片背景 | `#f9fafb` | 卡片、面板、列表项 |
| 主要文字 | `#1f2937` | 标题、正文 |
| 次要文字 | `#6b7280` | 描述、辅助信息 |
| 辅助文字 | `#9ca3af` | 日期、元信息 |

## 排版

| 属性 | 值 |
|------|-----|
| 字体 | Inter, system-ui, sans-serif |
| 首页标题 | clamp(45px, 8vw, 86px)，字重 800+ |
| 页面标题 | clamp(42px, 7vw, 66px) |
| 区块标题 | clamp(32px, 5vw, 54px) |
| 卡片标题 | 20-21px，字重 700 |
| 正文 | 15-18px，行高 1.55-1.7 |
| 元信息/标签 | 11-13px |

## 组件规范

### 圆角
- 大型面板/英雄区：34px
- 卡片：16-24px
- 按钮/标签：999px（胶囊形）
- 小图标容器：15px

### 阴影
- 深色模式：`rgba(0,0,0,0.40)` — 深色背景下使用更深阴影
- 浅色模式：`rgba(0,0,0,0.06)` — 浅色背景下使用轻柔阴影
- 悬停状态：增加蓝色光晕效果

### 过渡
- 卡片悬停：`transform 180ms ease, border-color 180ms ease, box-shadow 180ms ease`
- 上移距离：3px

## 响应式断点

| 断点 | 宽度 | 行为 |
|------|------|------|
| 桌面 | > 960px | 双列英雄区、三列卡片网格 |
| 平板 | ≤ 960px | 单列布局 |
| 手机 | ≤ 560px | 单列、按钮纵向排列、统计纵向排列 |
