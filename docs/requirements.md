# 开发需求文档

## 项目概述

基于 VitePress 的个人主页，部署在 GitHub Pages，用于展示个人信息、博客、项目、荣誉和论文。

## 功能需求

### 1. 首页
- 展示个人简介、姓名、标签语
- 6 个功能卡片，链接到各子页面
- 3 个统计数据展示

### 2. 博客系统
- 文章按发布日期倒序排列
- 支持文章标签（tags）分类展示
- 文章列表显示摘要（description）
- 新建文章只需在 `src/posts/` 下添加 `.md` 文件

### 3. 荣誉与奖项
- 独立页面展示荣誉列表
- 按日期排序
- 每条荣誉包含：标题、日期、颁发组织、描述

### 4. 论文与发表
- 独立页面展示论文列表
- 按日期排序
- 每篇论文包含：标题、日期、作者、期刊/会议、摘要、外部链接

### 5. 关于我
- 简短自我介绍
- 研究兴趣
- 技能
- 链接到荣誉和论文页面
- 联系方式

### 6. 主题
- 深色背景为主（VS Code 风格暗色主题）
- 浅蓝色（#60a5fa / #3b82f6）作为按钮、链接、标题的点缀色
- 支持一键切换白天/黑夜模式
- 默认为深色模式

## 非功能需求
- 部署在 GitHub Pages
- 静态网站，无需后端
- 用户无需懂代码即可添加内容（编辑 Markdown 文件即可）
