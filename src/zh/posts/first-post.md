# 第一篇博客文章

这是我的第一篇博客文章。在这里写下你的学习笔记、技术心得或项目记录。

## 如何写博客

1. 在 `src/zh/posts/` 目录下新建一个 `.md` 文件
2. 在文件开头填写 frontmatter（`---` 之间的内容）：
   - `title`: 文章标题
   - `date`: 发布日期（格式 YYYY-MM-DD）
   - `tags`: 标签列表（用方括号包裹，逗号分隔）
   - `description`: 文章摘要（显示在博客列表页）
3. 在 frontmatter 下方用 Markdown 写正文

## Markdown 基础语法

- `#` 开头表示标题（`##` 是二级标题）
- `**粗体**` 显示为 **粗体**
- `-` 开头表示列表项
- 用空行分隔段落
